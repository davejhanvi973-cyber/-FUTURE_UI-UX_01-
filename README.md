# Online Food Delivery React Application

## Project Overview
This project is a React-based Online Food Delivery application that allows users to browse food items, manage a cart, place orders, and receive order confirmation. It also includes an admin module for managing food items.

## Technologies Used
- React.js
- TypeScript
- Tailwind CSS
- LocalStorage (for data persistence)
- HTML & CSS

## Features
- Home page with navigation
- User login and signup
- Food menu with add-to-cart option
- Cart management
- Checkout process
- Order confirmation page
- Admin login and admin dashboard
- Hash-based navigation without React Router

## Project Structure

## How Navigation Works
This project uses `window.location.hash` for page navigation instead of React Router. The current page state is managed using React `useState`.

## Data Storage
- Default data is initialized using LocalStorage
- Cart items, users, and orders are stored locally in the browser
